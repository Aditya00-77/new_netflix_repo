# Netflix Data Analysis & Recommendation System

## 📌 Project Overview
This project explores the extensive Netflix catalog to uncover deep content trends and build intelligent movie and TV show recommendation systems. By analyzing release timelines, global production hubs, genre growth, and rating distributions, this analysis provides strategic insights into content distribution and leverages content-based filtering to enhance viewer engagement.

## 🚀 Key Features
* **Exploratory Data Analysis (EDA):** Dissected trends over time to trace the evolution of movies versus TV shows on the platform.
* **Text Mining & Feature Engineering:** Extracted, cleaned, and processed metadata, descriptions, and cast info using advanced NLP techniques.
* **Geospatial & Trend Analysis:** Visualized the top content-producing countries and mapped changing viewer preferences.
* **Recommendation System:** Developed a content-based filtering model utilizing TF-IDF Vectorization and Cosine Similarity.
* **Interactive Visualization:** Created dynamic visual plots to make complex viewing metrics easily understandable for stakeholders.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning & NLP:** Scikit-learn, NLTK
* **Data Visualization:** Matplotlib, Seaborn, Plotly
